
# 06. 从尾到头打印链表

* 输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。

* 输入：head = [1,3,2]
* 输出：[2,3,1]


```python
# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution(object):
    def reversePrint(self, head):
        """
        :type head: ListNode
        :rtype: List[int]
        """
        p = head
        result = []
        while(p):
            result.append(p.val)
            p = p.next
        return result[::-1]
```
